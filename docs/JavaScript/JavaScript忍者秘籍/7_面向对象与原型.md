# 第7章 面向对象与原型

隐式的函数参数 `this` 和 `arguments`。两者会被静默地传递给函数，并且可以像函数体内显式声明的参数一样被正常访问。

参数 `this` 表示**被调用函数的上下文对象**，而 `arguments` 参数表示函数调用过程中传递的所有参数。
